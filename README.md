# Семейный бюджет MVP

Streamlit-приложение для семейного бюджета с онбордингом на первом запуске.

## Что есть

- пошаговый onboarding wizard вместо пустых вкладок
- расчёт чистого дохода из gross по НДФЛ РФ 13% / 15%
- отдельное добавление нескольких ипотек и кредитов
- авторанжирование долгов: что гасить быстрее, что медленнее, что не трогать
- расчёт комфортного бюджета на жизнь из:
  - чистого дохода
  - обязательных расходов
  - выбранной стратегии
- стратегия распределяет свободный cash flow на:
  - жизнь
  - досрочку
  - накопления
- кнопка пересчёта стратегии в основном приложении
- кнопка полного сброса данных и возврата к онбордингу
- SQLite база

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
# Windows PowerShell:
# .venv\Scripts\Activate.ps1

pip install -r requirements.txt
streamlit run app.py
```

## Логика расчёта

1. Считается чистый доход.
2. Складываются обязательные расходы:
   - фиксированные
   - переменные обязательные
3. Свободный cash flow = чистый доход - обязательные расходы.
4. По стратегии свободный cash flow делится на:
   - бюджет на жизнь
   - досрочное погашение
   - накопления

## Стратегии

- `aggressive`: жизнь 45%, досрочка 40%, накопления 15%
- `balanced`: жизнь 60%, досрочка 25%, накопления 15%
- `soft`: жизнь 70%, досрочка 10%, накопления 20%
